Data Analysis & Visualization Projects

My recent work in data analysis and visualization includes two projects: the Suicide Rates Analysis Project and the Anti-Psych LAAD Project with Alkermes. In the Suicide Rates Analysis Project, I focused on creating an interactive dashboard designed to provide a deeper understanding of suicide trends across different demographics, including age, sex, and ethnicity. This dashboard, built with Holoviz Panel, allows users to filter data and observe trends within specific groups, allowing users to uncover patterns and correlations that might remain hidden in the raw data. Through this project, I honed my skills in data visualization, API integration, and building intuitive, user-focused interfaces, all critical for translating complex datasets into accessible, actionable insights.
